# 01 - The Magic Button (Starter Project) 🪄

Welcome to your first Web Development project at Tech Nexus! 

In this module, you will learn how the three core languages of the web work together to create an interactive experience.

## 🎯 What You Will Learn
* **HTML (`index.html`):** How to structure a webpage, create a card, and add a button.
* **CSS (`style.css`):** How to style elements, center them on the screen, and add hover effects.
* **JavaScript (`script.js`):** How to make the webpage interactive by listening for a click and changing the background color dynamically.

## 🚀 How to Run This Project
1. Download or clone this folder to your computer.
2. Open the folder.
3. Double-click the `index.html` file to open it in your web browser.
4. Click the button and watch the magic happen!

## 🛠️ Try It Yourself!
Once you get it working, try changing the code:
- Go into `index.html` and change the text on the button.
- Go into `style.css` and change the button's background color.
- Refresh your browser to see your changes live!
